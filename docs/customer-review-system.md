# Customer Review System

## Goal

Turn happy guests into public reviews and route unhappy feedback privately.

## Flow

1. Guest visits Papai
2. Staff sends review request by WhatsApp
3. AI asks if the experience was positive
4. Positive guests receive public review link
5. Negative or neutral feedback goes to a private feedback form
6. Manager sees weekly reputation report

## Dashboard Metrics

- review requests sent
- public reviews completed
- private feedback received
- unresolved issues
- weekly sentiment summary
