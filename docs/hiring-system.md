# Hiring System

## Goal

Turn the public hiring signal into a structured candidate pipeline.

## Flow

1. Candidate clicks WhatsApp or form link
2. Candidate chooses a role
3. Candidate submits name, phone, experience, and availability
4. AI creates a short candidate summary
5. Manager reviews candidates in the dashboard
6. Staff contacts selected candidates

## Candidate Fields

- full_name
- phone
- role
- experience
- availability
- ai_score
- status

## Status Values

- new
- reviewing
- contacted
- selected
- not_selected
