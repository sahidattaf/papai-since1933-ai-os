# WhatsApp Ordering Workflow

## Goal

Prepare Papai for WhatsApp-based menu questions, daily specials, reservation support, and future ordering.

## MVP Flow

1. Customer sends WhatsApp message
2. AI detects intent: reservation, menu, hiring, review, or general question
3. AI responds with approved template
4. System stores the interaction for manager review

## Future Upgrade

Add verified menu items, prices, pickup timing, delivery logic, and payment links once confirmed by Papai management.
